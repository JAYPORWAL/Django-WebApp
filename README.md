# Django WebApp 🚀

A simple Django app with full CI/CD pipeline using GitHub Actions and Docker.

## 📦 Stack Used

- Django
- Docker
- GitHub Actions
- DockerHub
- Minikube (or Local Docker)

## 🛠️ Run Locally

```bash
git clone https://github.com/JAYPORWAL/Django-WebApp.git
cd Django-WebApp
docker-compose up --build
